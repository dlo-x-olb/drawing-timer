Clear Stored Data

Data is stored in your web browser's local storage which can also be cleared by the browser. Albums are stored as a string of text containing a list of file names and a folder path, so deleting the data won't affect any personal files.

Create a New Album

An album points to a local folder on your device/computer and displays the images you have chosen here. For best performance choose images that have been scaled down to an appropriate size. Each album can support thousands of images.

The address of the folder that contains the images.
E.g. C:\Users\Public\Pictures\Poses
Windows tip: right click in the address bar of the choose file dialog and copy the address.
